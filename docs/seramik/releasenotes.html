Description

v0.1 (Initial Release)

  • Set of Django models for content analysis.

  • Developed serializers for model instances.

  • Implemented RESTful API endpoint for CRUD operations on InputRecords.

v0.1.1

  • Timestamp field in InputRecord model now uses Django’s DateTimeField.

  • Addition of related_name in ForeignKey and ManyToManyField for reverse queries.

  • Improved data validation across models.

For complete details on updates and changes, please refer to the documentation of each model.
