Models Overview in Mozaik
=========================

Mozaik integrates advanced AI models, each bringing unique capabilities to enhance the user experience.

ComfyUI
-------
- **Purpose**: Image generation and modification.
- **Advantages**: Intuitive GUI, 3D/video model support.
- [Read more about ComfyUI](models/comfyui)

GPT-4 and GPT-3.5
-----------------
- **Purpose**: Text generation and language understanding.
- **Advantages**: Multilingual, interactive.
- [Read more about GPT-4 and GPT-3.5](models/chatgpt)

Gemini Pro API
--------------
- **Purpose**: Multimodal content generation.
- **Advantages**: Versatile data processing, interactive experiences.
- [Read more about Gemini Pro API](models/gemini)

Stable Diffusion XL
-------------------
- **Purpose**: Image generation from text.
- **Advantages**: Customization, scalability.
- [Read more about Stable Diffusion XL](models/stablediffusion)

Control Net
-----------
- **Purpose**: Content moderation.
- **Advantages**: Real-time moderation, behavior analysis.
- [Read more about Control Net](models/controlnet)
